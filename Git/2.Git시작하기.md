# Git 시작하기
1. 로컬 리포지토리 작성(.git 폴더 생성)
    ```
     git init
    ```
    - 리포지토리
        - 압축파일
        - 트리파일
        - 커밋 파일
    - 인덱스 파일
    - 설정파일
2. Git리포지토리의 Copy를 작성
    ```
    git clone <리포지토리명>
    ```
3. 변경을 Stage에 추가한다
    ```
    git add .
    git add <파일명>
    git add <디렉토리명>
    ```
    - 커밋할 변경을 준비하는 단계
    - 압축파일 작성 → 파일명과 압축파일을 맵핑한 인덱스를 작성
