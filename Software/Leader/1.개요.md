   5. 품질관리

   7. 회고 (振り返り)

   2. <u>**기본설계**</u>
   3. <u>**상세설계**</u>
   4. <u>**제조**</u>
   5. <u>**단위 테스트**</u>

3. #### 제재(題材)로 삼고있는 프로젝트 계획

   - 음식점 리뷰 서비스
   - 시스템
     - Customer Web : 회원등록/탈퇴, 음식점 검색, 평판 등록, 즐겨찾기, 프리미엄 등록/탈퇴
     - Client Web : 점포 등록, 점포 정보 변경, 쿠폰 송신
     - Management Web : 공개 신청 승인, 회원 상세, 음식점 상세, 쿠폰 승인, 음식점 일람, 회원 일람
     - Batch : 월차과금, 월차회원집계
