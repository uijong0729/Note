# 1. 혼동행렬 (Confusion Matrix)
- 분류 문제에서 출력된 클래스 분류의 결과를 정리한 행렬(표)
- 진양성(True Positive, TP) : 모델이 양성으로 예측한 것 중 실제로 양성인 것
- 진음성(True Negative, TN) : 모델이 음성으로 예측한 것 중 실제로 음성인 것
- 위양성(False Positive, FP) : 모델이 양성으로 예측한 것 중 실제로 음성인 것
- 위음성(False Negative, FN) : 모델이 음성으로 예측한 것 중 실제로 양성인 것    

# 2. 정해율 (正解率, Accuracy)
- TP(진양성)와 TN(위음성)의 합을 전체 예측 수로 나눈 값
- TP + TN / TP + TN + FP + FN
- 언뜻 항상 좋은 지표로 보이지만, 모델의 퍼포먼스를 평가하기에 항상 우선도가 높은 지표는 아니다. 클래스 불균형이 심한 경우, 정해율이 높게 나올 수 있지만 모델의 성능이 좋지 않을 수 때문
- 예 : 990마리의 개와 10마리의 고양이가 있는 데이터셋에서, 모델이 모든 동물을 개로 예측한다면 정해율은 99%가 되지만 모델의 성능이 좋다고는 말하기 어렵다
