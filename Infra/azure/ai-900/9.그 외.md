1. Knowledge Mining(지식 마이닝)
- 기계 학습이나 자연어 처리 등의 AI 기술을 활용하여 조직 내 산재하는 부가가치가 높은 정보를 추출, 분석하는 프로세스
- 조직 내에 존재하는 다양한 데이터 소스에서 유용한 정보를 추출하여 비즈니스 인사이트를 얻는 것을 목표로 함
- 예 : 기업이 지식 마이닝을 사용하여 고객의 피드백이나 시장 트랜드를 분석해 제품이나 서비스를 개선

2. Document Intelligence
- Azure에서 제공하는 서비스 이름
- 청구서, 영수증, 명함 등의 문서에서 문자 정보를 취득하는 Azure Cognitive Services의 OCR기능 중 하나
- 특정 서류를 읽는 것에 특화
- 예 : 청구서나 영수증에서 데이터를 추출해 Document Intelligence를 활용하여 회계 프로세스를 자동화 

3. 異常検知(이상 감지)
- 정상적인 패턴에서 벗어난 데이터를 식별하는 프로세스

4. Computer vision(컴퓨터 비전)
- 화상분류(画像分類)
    - 흰 곰과 검은 곰을 구분하는 기술
- 물체검출
    - 사진 속에서 곰을 찾아내는 기술
- 손글씨로 된 문서 속의 문자를 특정하는 기술
- あなたの会社ではある**規格に基づいたペットボトルだけ**を回収する機械を作成しようとしています。その際に使うべき機械学習のモデルはどれでしょうか？

5. 光学的文字認識(Optical Character Recognition (OCR))
- マラソンの記録をとるために、ランナーのユニフォームに**書かれている番号を機械に読み取らせ**ようとしています。その際に使うサービスはどれでしょうか？
- 종업원의 경비정산에 있어서 영수증에서 그 금액을 읽는 모델

6. 음성합성
- 문자를 음성화해서 낭독하는 기술
- 音声合成の適用例として考えられるのは次のうちどれでしょうか？正しいものを二つ選択してください。
    - キーボードを使って入力された電話番号を読み上げる機能
    - ゲームの中でプレイヤーにキャラクターが話しかけるようにする機能

7. 異常検出(이상검출)
- 많은 데이터 중에 조금 다른 성질의 데이터를 찾아내는 기술
- 특수한 행동을 하는데 특화된 모델
- 보통이 아닌 카드 지불의 거동을 검출

8. 自然言語処理(자연언어처리)
- 문자 정보에서 감정을 분석
- SNS에 투고된 정보에서 감정을 분석하는 경우
- 번역
- 키-프레이즈 추출

9. 클러스터링
- 어느 특징에 근거하여 동물을 그룹화해 분류하는 기술 (기계학습)
