1. Knowledge Mining(지식 마이닝)
- 기계 학습이나 자연어 처리 등의 AI 기술을 활용하여 조직 내 산재하는 부가가치가 높은 정보를 추출, 분석하는 프로세스
- 조직 내에 존재하는 다양한 데이터 소스에서 유용한 정보를 추출하여 비즈니스 인사이트를 얻는 것을 목표로 함
- 예 : 기업이 지식 마이닝을 사용하여 고객의 피드백이나 시장 트랜드를 분석해 제품이나 서비스를 개선

2. Document Intelligence
- Azure에서 제공하는 서비스 이름
- 청구서, 영수증, 명함 등의 문서에서 문자 정보를 취득하는 Azure Cognitive Services의 OCR기능 중 하나
- 특정 서류를 읽는 것에 특화
- 예 : 청구서나 영수증에서 데이터를 추출해 Document Intelligence를 활용하여 회계 프로세스를 자동화 

3. 異常検知(이상 감지)
- 정상적인 패턴에서 벗어난 데이터를 식별하는 프로세스

4. Computer vision(컴퓨터 비전)
- 화상분류
- あなたの会社ではある**規格に基づいたペットボトルだけ**を回収する機械を作成しようとしています。その際に使うべき機械学習のモデルはどれでしょうか？
