1. 자동기계학습
- 동의어
    - AutoML
    - 自動ML
    - 自動機械学習宇
    - Automated Machine Learning
- 모델 개발 태스크를 자동화하는 프로세스
- 모델의 구축에 있어서 **시간을 요하는 반복적 작업을 자동화**하는 프로세스
- 자동기계학습이 예측해야할 라벨을 자동적으로 선택할 수는 없다
- 유저가 평가 척도를 정하면 그 지표로 모델을 평가해 가장 좋은 모델을 선택한다
- 수동으로 설정해줘야 하는 것
    - 라벨
    - 트레이닝 데이터
    - 지표(정확도, 정밀도, 재현율, 손실 등)
- 자동으로 수행되는 것
    - 시간을 요하는 반복적 작업
    - 지표에 부합하는 최적의 모델을 선택하는 것

2. Azure Machine Learning Designer
- Azure Machine Learning Workspace에서 기계학습 파이프라인을 구축하기 위한 드래그 앤 드롭 인터페이스
    - 기계학습 파이프라인의 구성요소
        - 1. 데이터
        - 2. 모델
        - 3. 컴포넌트 = 입력 + 출력 + 코드(메타데이터)
    - 자동차 가격을 예측하는 모델 작성 예
        - 입력 데이터 : 배기량, 마력, 연비, 제조년도, 브랜드, 색깔 등
        - 특징량의 선택 : 불필요한 정보를 걸러냄
        - 결손치의 처리
        - 데이터의 분할 : 데이터를 훈련용으로 쓸지, 평가(출력)용으로 쓸지 분할
        - 모델의 훈련 (또는 출력으로 직행)
        - 출력
    - 커스텀 코드를 쓸 경우 사용 가능한 언어는 Python과 R
- Azure Machine Learning Designer 만든 모델을 웹 서비스로서 엔드 유저에게 공개할 때 필요한 것
    - 인증키
    - REST 엔드포인트
- Azure Machine Learning Designer의 유용성
    - 1. 모델에 필요한 데이터의 클리닝이나 변환 등
    - 2. 기계학습 모델을 구축하기위한 트레이닝(Traning)
    - 3. 기계학습 모델의 배포(Deployment)와 관리

3. Azure Machine Learning
- 노 코드로 기계 학습 모델을 구축
- 기계학습 모델의 학습에서 구축까지 CUI로든 GUI로든 지원

4. Azure AI Service
- 음성인식이나 이미지분석 등 사전에 학습된 AI 모델을 이용한 어플리케이션을 구축하기 위해 제공되는 서비스

5. Azure AI Video Indexer
- 비디오 콘텐츠를 분석해 음성, 텍스트, 얼굴인식 등의 정보를 추출하는 Azure AI 서비스

6. Azure AI Bot Service
- 다양한 시나리오에 맞는 챗봇을 작성해 관리한다
- Teams, Facebook 등 다른 플랫폼과 통합하여 챗봇을 배포할 수 있다
- 固有表現認識 (NER) 등 **Azure AI Language** 서비스의 자연언어처리(NLP)와 결합하면 더 적합한 대답을 제공할 수 있다

7. Azure AI Vision
- 이미지나 동영상 내의 광학식문자인식(OCR), 이미지 분석, 비디오 분석을 제공한다
- 사전에 학습된 모델을 사용해 커스텀 트레이닝이 필요없다
- 이미지 분석(画像分析) : 이미지 내용에 근거하여 자동적으로 태그나 설명을 붙일 때 사용
- OCR : 이미지에서 텍스트 추출
- 画像分析 : 이미지의 물체를 검출해 분석
- 顔認識 : 이미지에서 얼굴을 인식해 분석
- ビデオ分析 : 동영상에서 사람이나 동물을 검출하고 추적
- 画像の説明（Image Description）: 이미지에 자동적으로 캡션을 부여

8. Azure AI Custom Vision
- 특정 물체나 이미지 카테고리를 검출하거나 분류하는 서비스
- 커스텀 트레이닝하여 특정 용도에 맞춰 모델을 구축할 수 있다
- 물체 검출 모델을 작성할 때는 멀티 라벨(マルチラベル) 또는 멀티 클래스(マルチクラス)의 분류 타입을 선택할 수 없다.

9. Azure AI Face
- 얼굴검출, 얼굴인식
- 감정이나 연령을 추정
- 얼굴의 유사성을 측정해 개인을 식별하거나 인증에 사용한다
- 顔検出 : 이미지에서 얼굴이라고 할 물체가 있는지 찾아내는 기능
- 顔検証 : 두 얼굴을 1대1로 비교하여 동일인인지 확인하는 기능
- 顔識別 : **데이터베이스**의 여러 얼굴 중 특정한 개인을 식별하는 기능
- 顔分析 : 이미지의 얼굴에서 연령, 감정, 안경 등 얼굴의 속성을 추출할 수 있는 기능
- 似た顔の検索 : 데이터에비스의 여러 얼굴 중 유사한 얼굴들을 검색하는 기능


10. Azure AI Speech(音声)
- Speech-To-Text
    - **음성을 텍스트**로 변환하고 실시간으로 번역
    - 온라인 회의나 웨비나(웹 세미나)에서 참가자에게 **받아적기(文字お越し)**를 제공
- Text-To-Speech : 텍스트를 음성으로 변환하는 서비스
- Speech Translation : 다른 국가 언어의 음성도 처리하거나 번역
- Intent Recognition(意図認識) : 텍스트나 음성에서 유저의 의도를 분석하는 기능

11. Azure AI Language
- 유저의 자연언어를 이해하고 의도를 분석
- 会話言語理解（CLU）
    - 문장이나 텍스트에서 감정, 키 문구(key phase), 엔티티를 추출
    - 엔티티 : AI가 고유의 개념으로 인식하거나 식별할 수 있는 정보의 단위
    - 유저가 자연스러운 회화문으로 입력한 내용에서 **의도와 엔티티**를 스스로 추출하는 기능을 제공
    - 종래의 Language Understanding(LUIS)를 대체하는 서비스
- Knowledge Base(ナレッジベース)를 활용해, 질문에 대한 대답을 생성한다
    - Knowledge Base : ai가 대답이나 처리의 근거로서 참조하는 전문지식 데이터베이스
- Text Analytics : 텍스트 데이터 내에서 키워드나 감정을 추출해 정보를 분석하는 서비스
- 質問応答
    - 문서나 Knowledge Base에서 질문에 관련된 대답을 제시
    - 문서나 Knowledge Base에서 질문에 관련된 부분을 자동적으로 추출해서 제시하는 기능
    - **유사한 질문에는 일관적으로 대답한다**
- Azure AI Translator(翻訳)
    - 다언어간 실시간 번역을 서포트
    - 번역 대상의 문맥을 이해하고 자연스러운 번역을 제공
- 固有表現認識（NER:Named Entity Recognition, 개체명인식）
    - 인명, 장소, 시간, 조직명 등 고유명사를 인식하고 분류하는 기능
- 質問応答機能（Question Answering）
    - Chit-Chat(おしゃべり)
        ・Professional（フォーマル）
        ・Friendly（親しみやすい）
        ・Enthusiastic（熱心な）

12. Azure AI Search
- 문서나 데이터에서 인덱스를 작성해 고속으로 검색을 제공
- 관련성에 따라 랭크를 매겨 검색결과의 품질을 향상
 - Knowledge Mining(ナレッジマイニング)
    - 기계학습이나 자연어처리의 기술을 이용해 다양한 데이터 소스에서 유익한 정보나 지식을 추출해 의사결정을 지원

13. Azure OpenAI Service
- OpenAI가 개발한 대규모언어모델(大規模言語モデル)을 사용
- 코딩 지원, 프로그램 생성, 챗봇, 대화형AI
- Azure OpenAI 서비스를 호출하기 위해 사용 가능한 툴
    - Azure SDK for python
    - Azure REST API : JSON 형식으로 제공
- GPT-4.0 
    - 챗봇에 집어넣으면 질문에 대해 자연스러운 대답이 가능
    - 파라미터 조정
        - 13-1. プレゼンス ペナルティ（Presence penalty）
            - 모델이 이미 언급한 내용을 반복적으로 언급하는 것을 방지하기 위한 패널티
        - 13-2. Temperature値
            - 0.0~1.0 사이의 값을 지정
            - 0.0에 가까운 값 : 안정적이고 신뢰성 높은 대답
            - 1.0에 가까운 값 : 창의적이고 랜덤성 높은 대답
        - 13-3. Max response (最大応答)
            - 모델이 한번에 생성할 수 있는 토큰의 상한을 설정해 대답을 간결하게 할지 장황하게 할지 조절할 수 있다
    - GPT계열은 GitHub Copilot에서 사용하는 모델 (예전엔 codex였음)
- Whisper
    - 음성인식(Automatic Speech Recognition：ASR) 모델
    - 음성을 문자로 변환하는 것에 특화
- DALL-E
    - 유저가 입력한 텍스트 내용에 맞는 이미지를 생성
    - 초현실주의 화가 살바도르 달리(Salvador Dalí)에서 따온 이름
- ファインチューニング (Fine-tuning)
    - 미리 학습된 대규모 언어 모델에 대해 자사 **독자 데이터를 추가로 학습**시켜 튜닝하는 방법
- グラウンディング（Grounding）
    - 모델이 학습하지 않았던 분야나 최신정보에 대해서도 대답할 수 있도록 **외부 데이터 소스에 접근**해 대답을 생성하는 방법
- コンテンツフィルタリング（Content Filtering）
    - AI모델이 생성한 출력을 스스로 체크해 부적절한 내용이 포함되어 있으면 차단하거나 수정하는 대책
