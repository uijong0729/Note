## 1. 자연어 처리
- 인간이 일상적으로 사용하고있는 자연언어를 컴퓨터에게 처리시키는 일련의 기술

## 2. 자연어 처리의 흐름
- 전처리
    - 형태소 분석
        : 명사, 동사, 형용사 등으로 단어를 분리하는 작업
    - 단어의 통일, 정규화
        : azure, Azure, AZURE 등으로 표현되는 단어를 하나로 통일하는 작업
    - 스탑워드의 제거
        : 있든 없든 의미가 변치 않는 단어(예: 조사, 접속사 등)를 제거하는 작업

- 특징량으로 변환
    - 벡터로 표현

## 3. 키 프레이즈(Key phrase) 추출
- 텍스트에서 중요한 프레이즈나 키워드를 자동적으로 특정해 추출하는 프로세스
- 순서
    - 1. 텍스트의 전처리 : 텍스트의 정규화, 토큰화, 스톱워드의 삭제등 
    - 2. 특징추출 : 각 단어나 프레이즈에 대해 중요성을 측정하기 위한 특징을 추출
    - 3. 중요도의 스코어링 : 추출된 특징에 기반해 각 단어나 프레이즈의 중요도를 스코어링
    - 4. 키 프레이즈의 선정 : 스코어가 높은 단어나 프레이즈를 키 프레이즈로서 선정

## 4. 고유 표현 추출
- 텍스트에서 특정한 카테고리에 속하는 이름이나 용어를 식별해 추출하는 프로세스
- 순서
    - 1. 텍스트의 전처리
    - 2. 고유표현의 식별
    - 3. 카테고리 분류
    - 4. 추출과 출력

## 5. 감정 분석
- 텍스트 데이터에서 감정이나 의견을 식별해 분류
- 순서
    - 1. 텍스트의 전처리
    - 2. 특징 추출
    - 3. 감정의 분류
    - 4. 결과의 해석
- 감정 분석의 구체 예
    - 소셜 미디어 분석 : 소셜 미디어에서 일반적인 의견이나 감정의 경향을 분석
    - 시장조사 : 제품 리뷰나 고객 피드백에서 소비자의 의견이나 만족도를 분석
    - 정치분석 : 정치적인 의논이나 캠페인의 감정적인 영향을 분석
    - 고객 서비스 : 고객 문의나 피드백을 분석해 서비스의 질을 향상 시킨다

## 6. 언어 모델
- 언어의 구조를 수학적으로 모델링하는 수법이며 단어, 구문, 문 등 언어 단위가 연속하는 확률을 추정해 단어의 패턴을 학습
- 순서
    - 훈련 데이터 준비 : 대량의 텍스트 데이터를 수집
    - 모델 구축 : 통계적 수법이나 기계학습 알고리즘을 사용해 언어의 통계적 특성을 포착하는 모델을 구축
    - 확률 추정 : 텍스트 내의 단어나 문의 출현 확률을 추정
