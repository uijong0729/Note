1. 자연어 처리
- 인간이 일상적으로 사용하고있는 자연언어를 컴퓨터에게 처리시키는 일련의 기술

2. 자연어 처리의 흐름
- 전처리
    - 형태소 분석
        : 명사, 동사, 형용사 등으로 단어를 분리하는 작업
    - 단어의 통일, 정규화
        : azure, Azure, AZURE 등으로 표현되는 단어를 하나로 통일하는 작업
    - 스탑워드의 제거
        : 있든 없든 의미가 변치 않는 단어(예: 조사, 접속사 등)를 제거하는 작업

- 특징량으로 변환
    - 벡터로 표현

3. 키 프레이즈(Key phrase) 추출
- 텍스트에서 중요한 프레이즈나 키워드를 자동적으로 특정해 추출하는 프로세스
- 순서
    - ①텍스트의 전처리 : 텍스트의 정규화, 토큰화, 스톱워드의 삭제등 
    - ②특징추출 : 각 단어나 프레이즈에 대해 중요성을 측정하기 위한 특징을 추출
    - ③중요도의 스코어링 : 추출된 특징에 기반해 각 단어나 프레이즈의 중요도를 스코어링
    - ④키 프레이즈의 선정 : 스코어가 높은 단어나 프레이즈를 키 프레이즈로서 선정
