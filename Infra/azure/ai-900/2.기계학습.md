## 1. 기계학습
- 데이터를 기반으로 모델이 알고리즘에 의해 스스로 배우고 최종적으로 최적인 모델을 작성하는 프로세스
- 훈련 데이터
    - 모델이 스스로 판단이나 예측할 수 있도록 모델을 훈련시키는 데 사용되는 데이터
    - 훈련 데이터가 편향(バイアス, Bias)을 일으키면 모델의 판단도 편향될 가능성이 높다
    - 과학습
        - 모델이 훈련 데이터에 너무 맞춰져서 미지의 데이터에 대한 예측정도(予測精度)가 낮아지는 현상
        - 과학습을 막기위해 데이터를 모델학습용과 평가용으로 나눈다
- 평가 데이터
    - 훈련 데이터로 훈련된 모델을 평가하고 튜닝하는데 사용된다
    - 훈련 데이터를 평가 데이터로서 사용하지는 않는다
- 테스트 데이터 : 최종적 평가를 위해 남겨둔 평가용 데이터 **(튜닝은 하지 않는다)**
## 2. 기계학습에서의 분류란
- ラベル付け : 모델을 훈련 시키기 전에, 어느 데이터가 어느 클래스에 속하는지를 할당하는 것
- 기계 학습에 있어 분류(Classification) 란 데이터에 대해 미리 정해진 라벨(카테고리)를 할당하는 분류 수법을 말한다. 
##### 2-1. 교사있음(정답 데이터가 있음)
- 특징(인풋)과 라벨(정답)이 있는 훈련 데이터를 사용해서 모델을 훈련시키는 방법
- 회귀 : 규칙이 있는 연속된 데이터라고 간주하고 이후의 데이터를 예측
    - ex: 과거 10년간의 소득을 분석하여 미래 상환 가능성을 평가하고 융자여부를 심사하는 경우
- 분류 : 데이터가 주어졌을 때 학습된 모델을 통해 어느 범주에 속한 데이터인지 판단하고 예측
    - ex1: 테스트 결과에 근거하여 수우미양가 그룹을 나누는 경우
    - ex2: 꽃가루 알레르기의 증상에 따라 경도, 중도로 그룹을 나누는 경우
    - ex3: 고객 정보를 보고 신용도를 확인해 융자여부를 심사를 하는 경우
##### 2-2. 교사없음(정답 데이터가 없음)
- 특징(인풋)만 있고 라벨(정답)이 없는 훈련 데이터를 사용해서 모델을 훈련시키는 방법
- 클러스터링
    - ex1: 서류에 포함되는 텍스트의 유사성에 기준하여 그룹을 나누는 경우
    - ex2: 유사한 구매성향을 갖는 사람들을 그룹화하는 경우
## 3. 기계학습모델
- 입력 데이터에 대해 예측이나 분류를 수행하는 컴퓨터 프로그램
- 키워드는 예측, 분류
## 4. 라벨
- 모델이 예측하고자 하는 목표로 하는 값 또는 결과 (=정답)
- 특징 : 메일의 본문, 메일의 제목
- 라벨 : 메일이 스팸인지 아닌지
