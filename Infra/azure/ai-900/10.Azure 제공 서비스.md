1. 자동기계학습
- 동의어
    - AutoML
    - 自動ML
    - 自動機械学習宇
    - Automated Machine Learning
- 모델 개발 태스크를 자동화하는 프로세스
- 모델의 구축에 있어서 시간을 요하는 반복적 작업을 자동화하는 프로세스
- 자동기계학습이 예측해야할 라벨을 자동적으로 선택할 수는 없다
- 자동기계학습은 규정된 지표(유저의 Input)에 근거하여 반복훈련을 할 수 있다

2. Azure Machine Learning Deginer
- Azure Machine Learning Workspace에서 기계학습 파이프라인을 구축하기 위한 드래그 앤 드롭 인터페이스
    - 기계학습 파이프라인의 구성요소
        - 1. 데이터
        - 2. 모델
        - 3. 컴포넌트 = 입력 + 출력 + 코드(메타데이터)
    - 자동차 가격을 예측하는 모델 작성 예
        - 입력 데이터 : 배기량, 마력, 연비, 제조년도, 브랜드, 색깔 등
        - 특징량의 선택 : 불필요한 정보를 걸러냄
        - 결손치의 처리
        - 데이터의 분할 : 데이터를 훈련용으로 쓸지, 평가(출력)용으로 쓸지 분할
        - 모델의 훈련 (또는 출력으로 직행)
        - 출력
    - 커스텀 코드를 쓸 경우 사용 가능한 언어는 Python과 R

3. Azure Machine Learning
- 노 코드로 기계 학습 모델을 구축
- 기계학습 모델의 학습에서 구축까지 CUI로든 GUI로든 지원

4. Azure AI Service
- 음성인식이나 이미지분석 등 사전에 학습된 AI 모델을 이용한 어플리케이션을 구축하기 위해 제공되는 서비스

5. Azure AI Video Indexer
- 비디오 콘텐츠를 분석해 음성, 텍스트, 얼굴인식 등의 정보를 추출하는 Azure AI 서비스

6. Azure AI Bot Service
- 다양한 시나리오에 맞는 챗봇을 작성해 관리한다
- Teams, Facebook 등 다른 플랫폼과 통합하여 챗봇을 배포할 수 있다

7. Azure AI Vision
- 이미지나 동영상 내의 광학식문자인식(OCR), 이미지 분석, 비디오 분석을 제공한다
- 사전에 학습된 모델을 사용해 커스텀 트레이닝이 필요없다
- 이미지 분석(画像分析) : 이미지 내용에 근거하여 자동적으로 태그나 설명을 붙일 때 사용

8. Azure AI Custom Vision
- 특정 물체나 이미지 카테고리를 검출하거나 분류하는 서비스
- 커스텀 트레이닝하여 특정 용도에 맞춰 모델을 구축할 수 있다

9. Azure AI Face
- 얼굴검출, 얼굴인식
- 감정이나 연령을 추정
- 얼굴의 유사성을 측정해 개인을 식별하거나 인증에 사용한다
- 얼굴 검출(顔検出)
    : 이미지에서 얼굴을 찾아내는 기능
- 얼굴 검증(顔検証)
    : 두 얼굴을 1대1로 비교하여 동일인인지 확인하는 기능
- 얼굴 식별(顔識別)
    : 데이터베이스의 여러 얼굴 중 특정한 개인을 식별하는 기능
- 유사 얼굴 검색(似た顔の検索)
    : 데이터에비스의 여러 얼굴 중 유사한 얼굴들을 검색하는 기능
- 얼굴 분석(顔分析)
    : 이미지의 얼굴에서 연령, 감정, 안경 등 얼굴의 속성을 추출할 수 있는 기능

10. Azure AI Speech(音声)
- Speech-To-Text
    - **음성을 텍스트**로 변환하고 실시간으로 번역
    - 온라인 회의나 웨비나(웹 세미나)에서 참가자에게 **받아적기(文字お越し)**를 제공
- Text-To-Speech : 텍스트를 음성으로 변환하는 서비스
- Speech Translation : 다른 국가 언어의 음성도 처리하거나 번역
- Intent Recognition(意図認識) : 텍스트나 음성에서 유저의 의도를 분석하는 기능

11. Azure AI Language
- 유저의 자연언어를 이해하고 의도를 분석
- 会話言語理解（CLU）
    - 문장이나 텍스트에서 감정, 키 문구(key phase), 엔티티를 추출
    - 엔티티 : AI가 고유의 개념으로 인식하거나 식별할 수 있는 정보의 단위
    - 유저가 자연스러운 회화문으로 입력한 내용에서 의도와 엔티티를 스스로 추출하는 기능을 제공
    - 종래의 Language Understanding(LUIS)를 대체하는 서비스
- Knowledge Base(ナレッジベース)를 활용해, 질문에 대한 대답을 생성한다
    - Knowledge Base : ai가 대답이나 처리의 근거로서 참조하는 전문지식 데이터베이스
