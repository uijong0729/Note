# 1. 화상분류(画像分類)
- 화상 내 물체 등을 특정 카테고리에 분류하는 작업
- 미리 정해진 카테고리가 기준이 되기 때문에 비교적 난이도가 낮다

# 2. 화상 분류의 프로세스
- 데이터 수집 : 분류하는 화상 데이터 셋을 수집
- 전처리 : 화상 사이즈 조절, 정규화 등
- 특징 추출 : 엣지 검출, 텍스쳐 해석 등을 사용해 특징량을 추출
- 모델 훈련 : 라벨링(정답을 정해둔)된 훈련 데이터 셋을 사용해서 모델을 훈련
- 평가와 튜닝 : 검증용 데이터를 사용해 모델의 퍼포먼스를 평가하고 필요에 따라 조절
- 예측 : 훈련된 모델을 사용해 새로운 미지의 화상을 분류

# 3. 화상분류 적용 예
- MRI 화상에서 종양을 발견
- 농지 화상에서 병충해를 발견

# 4. 물체 검출
- 사진 또는 동영상 내 특정한 물체를 식별해 그 위치를 바운더리 박스(경계상자)로 둘러싸는 프로세스

##### 4-1. 물체 검출의 수순
- 데이터의 수집과 전처리 : 대량의 화상 데이터 셋을 수집해, 전처리를 수행
- 라벨링 : 바운더리 박스와 라벨을 지정하는 작업
- 모델의 선택과 훈련 : 선택한 모델에 라벨링된 화상 데이터를 사용해 훈련
- 하이퍼 파라미터 튜닝
- 평가와 최적화 : 테스트 셋을 사용해 평가

##### 4-2. 물체 검출의 예
- 점포의 카메라를 이용해 진열대에서 재고의 변동을 검출
- 자동운전에 있어 주위 차량이나 신호기 등을 검출
- 스포츠에 있어 선수나 공의 위치나 움직임을 추적해 퍼포먼스 분석

# 5. OCR (Optical Character Recognition)
- 화상이나 문서 내 문자를 텍스트 데이터로 변환하는 기술
- 과거부터 있던 OCR기술과 구분하기위해 기계학습에 적용한 기술을 AI OCR이라고 부르기도 함
